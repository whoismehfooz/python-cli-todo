# 🐍 Python CLI To-Do App

A simple **Command Line Interface (CLI)** To-Do list application built with Python.
This project allows users to **add, view, and remove tasks directly from the terminal**.

It was built as a beginner project to practice **Python fundamentals and CLI program design**.



## ✨ Features

✔ Add new tasks
✔ View all current tasks
✔ Remove tasks by number
✔ Interactive terminal menu
✔ Continuous running until user exits


 🖥 Example Interaction


Welcome to To-Do App 💓

1. Add Task
2. View Tasks
3. Remove Task
4. Exit

Enter your choice: 1
Enter a task to add: Learn FastAPI

Enter your choice: 2
